-사용조건-
1. 크롬이 반드시 필수적으로 설치되어 있어야함.
2. 현재는 엑스엘게임즈 계정만 가능함.

- 사용법 -

1. 폴더 내에 client.exe , main.exe , ID.json 3파일이 한곳에 있어야 함.
2. client.exe 를 실행하여 현재 1~5번 배열에 존재하는 계정 정보를 확인가능
3. ID , PW 칸에 원하는 ID/PW 를 작성하고 우측에 1~5번 버튼을 클릭하여 원하는 위치를 선택한 뒤 저장 버튼을 눌러 저장할 수 있음.
4. ID/PW를 삭제하고자 한다면 우측 1~5번 버튼을 선택한 뒤 삭제버튼을 누르면 삭제됨.
5. client로 Json을 수정했으면 main.exe를 실행
